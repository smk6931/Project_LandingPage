=== 프로젝트 개요 ===

본 프로젝트는 자사의 크몽 및 외주 의뢰(아임웹 기반 웹사이트 제작) 서비스를 홍보하고
고객의 의뢰 내용을 접수받아 데이터베이스에 저장하는 랜딩페이지 구축을 목표로 함.

=== 기술 스택 ===

1. Frontend
   - React (Vite 사용 권장)
   - 모던한 UI/UX 적용 (애니메이션, 반응형 웹)
   - 컴포넌트 기반 설계

2. Backend
   - FastAPI (Python)
   - 가상환경(venv) 필수 사용
   - RESTful API 구축

3. Database
   - PostgreSQL (Supabase 활용 검토 중)
   - 고객 문의 및 제안서 데이터 저장

=== 주요 기능 ===

1. 회사 소개 (Hero Section)
   - 강렬한 첫인상 및 서비스 아이덴티티 전달

2. 포트폴리오 소개
   - 아임웹 및 기존 구축 사례 갤러리

3. 제안서 요청 (Contact Form)
   - 고객 정보 입력 (이름, 연락처, 의뢰 내용 등)
   - 제출 시 DB 저장

4. 관리 기능 (추후 확장)
   - 접수된 의뢰 목록 조회

=== 개발 일정 계획 ===

1. 1단계: 프로젝트 세팅 및 DB 설계
   - 리포지토리 초기화
   - Supabase/PostgreSQL 스키마 확정

2. 2단계: 백엔드 개발 (FastAPI)
   - API 엔드포인트 구현 (문의 등록)
   - DB 연동

3. 3단계: 프론트엔드 개발 (React)
   - UI 퍼블리싱 (메인, 포트폴리오, 문의폼)
   - API 연동

4. 4단계: 배포 및 최적화
   - 클라우드 서버 배포 (AWS or Oracle)
