=== 프론트엔드 초안 개발 명세서 ===

작성일: 2026-01-20
목표: 디자인 팀장 및 고객에게 전문성을 어필할 수 있는 프리미엄 랜딩페이지 구축

=== 디렉토리 구조 ===

Front/src/
  components/ (재사용 컴포넌트)
    Button.jsx (메인/서브 버튼)
    PortfolioCard.jsx (이미지/태그 포함된 카드)
  sections/ (페이지 섹션)
    Hero.jsx (메인 상단)
    Portfolio.jsx (포트폴리오 갤러리)
    Process.jsx (작업 진행 과정)
    Guidelines.jsx (작업 규정)
    QuoteForm.jsx (견적 문의)
  App.jsx (전체 레이아웃 조립)
  index.css (전역 스타일 및 애니메이션)

=== 섹션별 상세 기능 ===

1. Hero Section (메인)
   - 강렬한 인상을 주는 다크 테마 + 그라디언트 배경
   - 떠다니는 Orb(구체) 애니메이션 적용
   - 주요 통계(완료 프로젝트, 만족도 등) 노출
   - 견적 문의 및 포트폴리오 바로가기 버튼 배치

2. Portfolio Section
   - 실제 구축 사례를 보여주는 6개 카드형 레이아웃
   - 마우스 오버 시 확대 및 '보러가기' 오버레이 효과
   - 프로젝트별 태그(Tech Stack) 표시
   - 클릭 시 해당 외부 링크로 이동

3. Process Section
   - 6단계 작업 공정 타임라인 시각화
   - 상담 -> 기획 -> 개발 -> 테스트 -> 배포 -> 유지보수
   - 각 단계별 아이콘 및 설명 제공
   - 순차적 등장 애니메이션 적용

4. Guidelines Section
   - 고객이 궁금해하는 핵심 정책 안내
   - 수정 횟수, 제작 기간, 결제 조건, 준비 자료 등
   - 명확한 아이콘과 리스트 형태로 가독성 확보

5. QuoteForm Section
   - 고객 정보를 수집하는 입력 폼
   - 이름, 연락처, 이메일, 회사명
   - 프로젝트 유형(아임웹/커스텀 등) 및 예산 범위 선택
   - 유효성 검사 및 제출 시 알림 기능

=== 디자인 시스템 ===

1. 컬러 팔레트
   - 배경: Deep Navy (#0f172a) 기반의 다크 모드
   - 포인트: Violet & Blue Gradient (#667eea ~ #764ba2)
   - 텍스트: White & Light Gray

2. 인터랙션
   - Scroll Reveal: 스크롤 시 요소가 아래에서 위로 부드럽게 등장
   - Glassmorphism: 반투명한 유리 질감의 카드 UI
   - Hover Effects: 버튼 및 카드에 마우스 호버 시 입체감 부여
